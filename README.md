# My Project —— 多人在线卡牌对战游戏（全栈）

独立开发前后端完整栈，基于 Unity 2019.4.6f1 + Python Flask 实现多人在线卡牌对战游戏。

## 技术栈

**客户端：** Unity 2022.3.43f1c1 · C# · Unity UI · Unity Netcode  
**服务端：** Python · Flask-RESTX · MySQL · Redis · JWT · BCrypt

---

## 客户端（Unity）

### 核心系统

- **UI 窗口管理框架**：WndManager + BaseWnd 抽象窗口基类，支持账号/大厅/房间/匹配/对局 5 类窗口
- **网络层**：HTTPHandler 封装 RESTful 请求 + LobbyNet 联机事件回调
- **登录系统**：注册/登录/自动登录，JWT Token 持久化
- **房间系统**：创建房间、加入房间、双端同步房间状态
- **匹配系统**：快速匹配，匹配成功自动进入对局

## 服务端（Python Flask）

### 功能模块

- **用户系统**：注册/登录（BCrypt 加密 + JWT Token 鉴权）
- **房间系统**：创建房间、加入/离开、房间状态机
- **对战逻辑**：出牌提交、回合切换、胜负判定、Heartbeat 保活
- **战绩系统**：每局结束写入 MySQL，支持按用户查询
- **邮件系统**：发送/接收/标记已读/删除
- **好友系统**：添加/删除好友、查询好友列表
- **头像上传**：Base64 图片上传，支持查询接口
- **邮件验证码**：对接 SMTP 服务器，用于密码重置

### 数据库设计

4 张表：`user` · `room` · `mail` · `friend`

初始化：`python init_db.py`
